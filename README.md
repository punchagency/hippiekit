# Hippiekit

Your One-Stop Eco Shop

## Project Structure

```
hippiekit/
├── client/          # Vite + React frontend
│   ├── src/
│   ├── public/
│   ├── package.json
│   └── vite.config.ts
└── server/          # Backend (to be set up)
    └── .gitkeep
```

## Getting Started

### Client (Frontend)

```bash
cd client
npm install
npm run dev
```

The frontend will run on `http://localhost:5173`

## Features

- ✅ Onboarding flow (4 screens)
- ✅ Authentication pages (Sign In, Sign Up, Reset Password)
- ✅ OTP Verification
- ✅ Fluid typography with Tailwind CSS v4
- ✅ Mobile-first design (440px × 990px)
- ✅ Custom fonts (Poppins, Lato, Segoe Print)

## Tech Stack

### Frontend
- React 18
- TypeScript
- Vite
- Tailwind CSS v4
- React Router DOM
- React Hook Form + Zod
- shadcn/ui components
